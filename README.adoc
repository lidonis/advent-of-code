= Advent of code

Code for _advent of code_ in Kotlin

== Run Code

[source,shell]
----
 ./gradlew run --args="-y2020 -d1"
----

== Native Image

A native image can be build with GraalVM

[source,shell]
----
./gradlew nativeImage
----

The native image wil be available at build/graal/advent-of-code
